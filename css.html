Принципы быстродействия

Размещение ссылок на стили наверху страницы.
Размещение JavaScript внизу (насколько это возможно).
Делать как можно меньше HTTP-запросов.
Раздавать ресурсы с нескольких доменов, можно увеличить количество ресурсов, которые браузер будет загружать параллельно.
Никогда не раздавайть CSS с отдельного домена, поскольку это приводит к задерживающим отрисовку DNS-запросам.
Вставляйть CSS в HTML-код как можно раньше, чтобы браузер мог скачать его и заняться другими вещами.
Собирайть CSS в один файл. Браузер все равно скачает весь CSS, поэтому гораздо лучше просто собрать все, в один HTTP-запрос.
Сжать его с помощью gzip и минифицировать, чтобы браузеру пришлось загружать меньше.
Закэшировать все, что можно, чтобы весь этот процесс происходил как можно реже.

Селекторы

селектор1 + селектор2 стили приминяются к соседнему селектору селектор2

селектор1 > селектор2 > селектор3 стили приминяются к дочернему селектору селектор3

селектор:псевдокласс псевдокласс добавляется к селектору c помощью символа :

a:link выбирает ещё не посещённые ссылки

a:visited выбирает посещённые ссылки

a:active выбирает активные ссылки (кнопка мыши зажата на ссылке)

Flexbox

justify-content - выравнивает элементы горизонтально

flex-start: Элементы выравниваются по левой стороне контейнера.

flex-end: Элементы выравниваются по правой стороне контейнера.

center: Элементы выравниваются по центру контейнера.

space-between: Элементы отображаются с одинаковыми отступами между ними.

space-around: Элементы отображаются с одинаковыми отступами вокруг них.

align-items(align-self) - выравнивает элементы вертикально

flex-start: Элементы выравниваются по верхнему краю контейнера.

flex-end: Элементы выравниваются по нижнему краю контейнера.

center: Элементы выравниваются вертикально по центру контейнера.

space-between: Элементы отображаются на базовой линии контейнера.

space-around: Элементы растягиваются, чтоб заполнить контейнер.

flex-direction направление, в котором будут расположены элементы в контейнере

row: Элементы размещаются по направлению текста.

row-reverse: Элементы отображаются в обратном порядке к направлению текста.

column: Элементы распологаются сверху вниз.

column-reverse: Элементы распологаются снизу вверх.

order порядка отображения элементов в контейнере

flex-wrap отображение елементов в ряду

nowrap: Размеры элементов устанавливаются автоматически, чтоб они поместились в один ряд.

wrap: Элементы автоматически переносятся на новую строку.

wrap-reverse: Элементы автоматически переносятся на новую строку, но строки расположены в обратном порядке.

flex-flow два в одном flex-direction и flex-wrap

align-content указывает, как несколько рядов должны отделяться друг от друга

flex-start: Ряды группируются в верхней части контейнера.

flex-end: Ряды группируются в нижней части контейнера.

center: Ряды группируются вертикально по центру контейнера.

space-between: Ряды отображаются с одинаковыми расстояниями между ними.

space-around: Ряды отображаются с одинаковыми расстояниями вокруг них.

stretch: Ряды растягиваются, чтоб заполнить контейнер равномерно.

Text

white-space - управляет пробелами и переносами

nowrap - отображает весь текст одной строкой без переносов;

pre - сохраняет пробелы и переносы как в исходном коде аналогично тегу pre;

pre-wrap - работает как значение pre, но добавляет автоматические переносы, если текст не помещается в контейнер;

normal - режим по умолчанию.
